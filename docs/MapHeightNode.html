Class: MapHeightNode

MapHeightNode(parentNode, mapView, location, level, x, y, material, geometry)

new MapHeightNode(parentNode, mapView, location, level, x, y, material, geometry)

Represents a height map tile node that can be subdivided into other height nodes. Its important to update match the height of the tile with the neighbors nodes edge heights to ensure proper continuity of the surface. The height node is designed to use MapBox elevation tile encoded data as described in https://www.mapbox.com/help/access-elevation-data/

Class: MapBoxProvider

MapBoxProvider(apiToken, id, mode, format, useHDPI)

new MapBoxProvider(apiToken, id, mode, format, useHDPI)

Map box service tile provider. Map tiles can be fetched from style or from a map id. API Reference - https://www.mapbox.com/
Parameters:
Name Type Description
apiToken string Map box api token.
id string Map style or mapID if the mode is set to MAP_ID.
mode number Map tile access mode.
format string Image format.
useHDPI boolean
